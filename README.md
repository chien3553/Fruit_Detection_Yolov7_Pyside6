## Run 
```bash
  python main.py
```

## Function

1. Support image/video/webcam as input
2. Change model
3. Change IoU
4. Change confidence
5. Play/Pause/Stop
6. Show statistics 
8. Export detected image/video(webcam)
9. Show FPS
10. Show results in LCD(i2c) via arduino
