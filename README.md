## Run 
```bash
  python main.py
```

## Feature

1. Support image/video/webcam as input
2. Import/Change model
3. Change IoU
4. Change confidence
5. Run/Pause/Stop
6. Show statistics 
7. Export detected image/video(webcam)
8. Show FPS
9. Show results in LCD(i2c) via arduino
